# Q Learning in Grid World

## 環境
- python 3.6

## 各ファイルについて
- grid_world.py:グリッドワールド
- qlearning_agent.py:Q学習エージェント
- main.py:実験実行用のプログラム

## 実行方法

1. このリポジトリをクローン

2. main.pyを実行

```
python main.py
```
